## ❓ What is intifadah?

intifadah is a simple tool made for people who want to show their support for Palestine. It lets you add a twibbon—a small overlay image—to your social media profile pictures. This helps you spread awareness in an easy and accessible way. You don’t need any technical skills to use intifadah; the app guides you through each step.
